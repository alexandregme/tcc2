Primeiramente para esse projeto foram realizadas as pesquisas bibliográficas. Trabalhos de outros autores que de alguma forma se relacionam com a monografia foram estudados com a finalidade fornecer uma base de conhecimento sobre o tema, como por exemplo,  o entendimento de todo o processo de descoberta do conhecimento e suas técnicas de mineração de dados. Com isso foi possível montar os capítulos de revisão bibliográficas abordando os principais conceitos utilizados durante o desenvolvimento.

No segundo momento, o desenvolvimento trata-se da aplicação dos passos do processo de KDD sobre os Microdados do Censo da Educação Superior. Foram executados os seguintes passos: seleção de dados, pré-processamento, transformação, mineração de dados e interpretação.Na seleção de dados foi definida a fonte dos dados. No pré-processamento removemos principalmente erros relacionados com a falta de informação. Na transformação os dados foram enriquecidos. Partindo para a mineração de dados, os algoritmos de influencias, previsão, clusterização, e análise de cesta de compras foram aplicados. Finalmente, na interpretação dos dados os gráficos e análises encontradas são apresentados.
